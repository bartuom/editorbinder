# Downloads

Use the GitHub Release assets in this order:

1. `EditorBinder-<version>-win-x64.exe`
   - Best default for most Windows users.
   - One file to download and run.
   - Stores user tools and settings in `%APPDATA%\EditorBinder`.

2. `EditorBinder-<version>-win-x64.zip`
   - Portable/debug package.
   - Contains `EditorBinder.exe`, helper BAT files, docs, license files, and
     bundled runtime files.
   - Use `enable_portable_mode.bat` if you want tools and settings next to the
     app in the local `data` folder.

3. `EditorBinder-source-<version>.zip`
   - Developer/source package.
   - Contains `src\`, docs, tests-adjacent examples, and BAT launch files for a
     local Python install.
   - This is not the recommended download for ordinary Windows users.

GitHub also shows automatic `Source code (zip)` and `Source code (tar.gz)`
archives. Those are generated by GitHub from the repository and are mainly for
developers.
