# Support Policy

EditorBinder is community-supported open source software.

## Getting Help

- Use GitHub Issues for reproducible bugs and focused feature requests.
- Include your EditorBinder version, Python version, Windows version, and the
  relevant Unreal Engine version when reporting Unreal Python workflow issues.
- Do not attach private Unreal project files unless you have already removed
  proprietary content and are comfortable sharing them publicly.

## Scope

The project can help with EditorBinder bugs, import/export behavior, packaging,
and documentation. Unreal Engine Python API differences should be reported with
the copied script, the Unreal Output Log error, and the engine version.

## Security

Please do not open public issues for vulnerabilities. Follow `SECURITY.md`
instead.
